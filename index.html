Parity

Open source trading infrastructure

Parity gives you tools for building trading systems. Whether you are designing trading algorithms at an investment bank, researching market microstructure at a university, or just interested in how modern markets work, have a look at what we offer.

Parity

JVM

Parity is a software platform for running a trading venue. You can use Parity to set up your own exchange, as a test environment for your trading algorithms, or as a research environment for your academic study.

Philadelphia

JVM

Philadelphia implements the FIX protocol, the lingua franca of financial technology, on the JVM. Designed to exhibit low and predictable latency, it represents the next generation in open source FIX engines for the JVM.

  • Code
  • API

Juncture

JVM

Juncture implements network protocols for connecting to trading venues, such as NASDAQ, on the JVM. You can use Juncture for both live market connectivity and historical market data analysis.

  • Code
  • API

Nassau

JVM

Nassau implements NASDAQ transport protocols on the JVM. Originally developed for the NASDAQ stock exchange, these protocols are suitable for all applications that demand low latency and high reliability.
